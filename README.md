# @hirakawa/pptx

A TypeScript library for creating and manipulating PowerPoint (PPTX) files.

## Installation

```bash
npm install @hirakawa/pptx
```

## Usage

```typescript
import { hello } from '@hirakawa/pptx';

console.log(hello('World'));
```

## Development

### Install dependencies

```bash
npm install
```

### Build

```bash
npm run build
```

### Test

```bash
# Run tests
npm test

# Run tests with UI
npm run test:ui

# Run tests with coverage
npm run test:coverage
```

### Code Quality

```bash
# Lint
npm run lint

# Format
npm run format

# Type check
npm run typecheck
```
